# LexVision 2.0 - Webapplikation für die Verwaltung und KI -gestützte Analyse von Beweismitteln

## Beschreibung
Das Projekt erweitert die bestehende Anwendung "LexVision" um lokale KI-Funktionen zur schnellen Analyse von Text- und Bildbeweismitteln. Ziel ist eine effiziente, sichere und datenschutzkonforme Unterstützung juristischer Auswertungen. 

Das Ganze findet im Rahmen der Diplomarbeit an der HAK Imst statt.


## Ziel der Diplomarbeit:
LexVision soll um lokale KI-Funktionen erweitert werden, um Text- und Bildbeweismittel effizient auszuwerten. Durch ein modernes, erweiterbares System mit verbesserter Oberfläche sollen Sicherheit, Datenschutz und Analysequalität deutlich gesteigert werden.

## Technologien
- Backend: Python Flask
- Frontend: JavaScript / React
- Datenbank: MySQL, FAISS
